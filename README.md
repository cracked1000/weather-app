# Weather Application

A full-stack weather app built with **Spring Boot** (backend) and **React** (frontend).  
Authentication is handled by **Auth0**, and only users from **Fidenz Technologies** can log in.  

## Features  
- Real-time weather data (OpenWeather API)  
- Secure login via Auth0 (restricted to Fidenz users)  
- React frontend + Spring Boot backend


## Setup
- Download or clone this repo
- Open the backend in the javaIDE
- Open the frontend in any code editor (visual studio code)
- Run both application and navigate to the link in frontend
- Fill out the user details to login
